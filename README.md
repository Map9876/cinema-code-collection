## 二、核心编码规则  

### 1. 电影院编码（8位）  
**结构**：`省码(2) + 地市码(2) + 影院序号(3) + 属性码(1)`  
**属性码分类**：  
| 代码 | 类型                | 适用范围 |  
|------|---------------------|----------|  
| 0    | 市区电影放映队      | 城市     |  
| 1    | 市区专业电影院      | 城市     |  
| 5    | 农村电影放映队      | 农村     |  
| 6    | 农村专业电影院      | 农村     |  
*(完整分类见GY/T 207-2005附录B)*  

### 2. 影片编码（12位）  
**结构**：  
```
1-2位：国家/地区 | 3位：版本 | 4-7位：颁布年代 | 8-12位：排次号
```  
**国家/地区代码示例**：  
| 代码 | 国家/地区 | 代码 | 国家/地区 |  
|------|-----------|------|-----------|  
| 001  | 中国      | 071  | 芬兰      |  
| 002  | 香港      | 072  | 挪威      |  

**影片版本代码**：  
| 代码 | 版本类型       |  
|------|----------------|  
| 0    | 观摩影片       |  
| 1    | 普通版         |  
| 2    | 普通立体版     |  
| 3    | 巨幕版         |
